表格标签(重要):

表格的主要作用:
用于显示, 展示数据, 因为它可以让数据显示的非常的规整,可读性非常好. 特别是后台展示数据的时候, 能够熟练运用表格, 就显得尤为重要. 能把数据表现的很有条理
表格不是用来布局的, 而是用来展示数据的.
表格的基本语法:
table 标签 用于定义表格.
tr 标签 用于定义表格中的行, 必须嵌套在 table 标签中.
td 标签 用于定义表格中的单元格, 必须嵌套在 tr 标签中.
th 标签 标识表格的表头部分, 必须嵌套在 tr 标签中. 表头单元格 常用于表格第一行, 突出重要性, 表头单元格里面的问题会加粗居中显示
thread 标签 表格的头部区域, 位于表格的第一行.
tbody 标签 表格的主体区域, 用于放数据.

表格属性:
align : 规定表格相对周围元素的对齐方式. 属性值: left, center, right .
border : 规定表格单元是否拥有边框,默认为 "", 表示没有边框.
cellpadding : 规定单元边沿与其内容之间的空白, 默认像素值为 1 .
cellspacing : 规定单元格之间的空白, 默认像素值为 2 .
width : 表格的宽度, 属性值: 像素值 或 百分比.
height : 表格的高度, 属性值: 像素值 或 百分比.
合并单元格:
rowspan : 跨行合并单元格, rowspan="合并单元格的个数"
colspan : 跨列合并单元格, colspan="合并单元格的个数"
目标单元格(写合并代码的单元格):
跨行: 最上侧单元格为目标单元格, 写合并代码
跨列: 最左侧单元格为目标单元格, 写合并代码

无序列表:

ul: 表示 无序列表, 一般会以项目符号呈现列表项, 而列表项使用 li 标签定义
ul 标签 中只能嵌套 li 标签, 直接在 ul 标签中输入其他标签 或者 文字的做法是不被允许的.
li 标签 相当于一个容器, 可以容纳所有元素.

表单标签:

label标签:

label 标签 为input 元素定义标注(标签).
label 标签用于绑定一个表单元素, 当点击 label 标签内的文本时,浏览器就会自动将焦点(光标)或者选择对应的表单元素上, 用来增加用户的体验.
核心: label 标签的 for属性 应当与相关元素的id 属性相同.
示例: 当点击用户名时,即可向文本框中输入内容.

表单域

完整的表单通常由表单域, 表单控件, 提示信息三部份构成.
form 标签 用于定义 表单域 , 以实现用户信息的收集和传递.
form 会把它范围内的表单元素信息提交给服务器.
属性:
action : 用于指定接收并处理表单数据的服务器程序的url地址.
method : 用于设置表单数据的提交方式, 其取值为 get 或 post .
name : 用于指定表单的名称, 以区分同一个页面总多个表单域

表单元素

input 输入框 :

checked属性: 规定此input元素首次加载时应该被选中. 主要针对于 单选按钮 和 复选框 , 在加载页面时, 默认选中某个表单的值 .
