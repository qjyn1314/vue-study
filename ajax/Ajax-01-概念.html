XML

xml 可扩展标记语言
xml被设计用来传输和存储数据
xml和html类似, 不同的是html中都是预定义标签, 而xml中没有预定义标签.

AJAX

优点

无需刷新页面与服务器进行通信.
允许你根据用户事件来更新部分页面内容.

缺点

没有浏览记录, 不能回退.
存在跨域问题.

HTTP

超文本传输协议, 协议详细规定了浏览器和万维网服务器之间互相通信的规则.

请求报文:

行: 请求方式(GET, POST) + url + http版本 (http/1.1)
头:
Host, Cookie, Content-type(请求类型),
空行
体: get请求为空,post请求可以不为空.

响应报文:

行: http/1.1 200 ok

空行:
